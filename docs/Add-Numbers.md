# Add-Numbers

## SYNOPSIS
Returns the sum of two numbers

## SYNTAX

```
Add-Numbers [[-Number1] <Int32>] [[-Number2] <Int32>]
```

## DESCRIPTION
Will add two numbers together

## EXAMPLES

### EXAMPLE 1
```
Add-Numbers 1 1
File.doc
```
