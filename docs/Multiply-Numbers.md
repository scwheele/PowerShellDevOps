# Multiply-Numbers

## SYNOPSIS
Returns the multiples of two numbers

## SYNTAX
a
```
Multiply-Numbers [[-Number1] <Int32>] [[-Number2] <Int32>]
```

## DESCRIPTION
Will multiply two numbers

## EXAMPLES

### EXAMPLE 1
```
Multiply-Numbers 1 1
```
